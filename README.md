# 🏏 ODI Cricket Analytics Dashboard

An interactive Streamlit dashboard for analyzing Men's ODI Cricket match data from 2002–2023. This project combines exploratory data analysis (EDA), cricket performance analytics, and interactive visualizations to uncover insights about teams, players, venues, and match outcomes.


## 🎯 Project Objectives

* Analyze ODI cricket matches from 2002–2023
* Explore team performance trends
* Evaluate batting and bowling statistics
* Study toss impact on match outcomes
* Compare teams across venues and years
* Generate interactive cricket insights using Streamlit


## 📊 Dashboard Features

### 🏆 Team Analytics

* Team win percentage
* Toss impact analysis
* Head-to-head comparisons
* Team performance trends

### 🎯 Batting Analytics

* Top run scorers
* Batting averages
* Strike rate analysis
* Over-by-over scoring trends

### 🎳 Bowling Analytics

* Top wicket takers
* Economy rate comparison
* Bowling performance analysis
* Dismissal type distribution

### 📈 Trend Analysis

* Yearly ODI trends
* Average score progression
* Venue-based analysis
* Historical performance insights

### 🇵🇰 Pakistan Focus

* Pakistan team performance analysis
* Batting and bowling trends
* Match outcome statistics

## 🔧 Technologies Used

## 🚀 How to Run

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Dashboard

```bash
streamlit run app.py
```
